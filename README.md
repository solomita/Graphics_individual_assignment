# Interactive 3D Chair Viewer

### Prerequisites
- **Node.js** 
- **NPM** 

### Installation & Running

1. **go to the threejs directory:**
   ```bash  
   cd threejs
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash  
   npm run dev
   ```

4. **Open your browser** and go to:
   ```
   http://localhost:5173/
   ```
